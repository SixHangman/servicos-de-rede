HTTP e HTTPs

A sigla HTTPs significa Hypertext Transfer Protocol Secure, ou Protocolo de Transferência de Hipertexto Seguro. O termo é muito parecido com a conhecida sigla HTTP, mas com a diferença do “s” de segurança incluído no final

HTTP e HTTPS: como saber se um site é seguro? Na maioria dos navegadores web, como o Google Chrome ou o Microsoft Edge, por exemplo, a presença de um cadeado ao lado do endereço indica o uso do protocolo HTTPS

cinco etapas: 1-hospede seu site com um IP dedicado; 2-compre um certificado SSL; 3-solicite o certificado SSL; 4-instale o certificado; 5-atualize seu site para ativar o HTTPS.

WEBSERVER

Um web server é um sistema computacional que hospeda e fornece acesso aos conteúdos e aplicações através da internet. Geralmente contratado como um serviço, esse servidor recebe e processa as solicitações feitas por navegadores através de protocolos de rede como o HTTP ou HTTPS.
